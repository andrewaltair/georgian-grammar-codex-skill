# Normative Orthography

Use this reference when Georgian output includes spelling, hyphenation, joined/separate writing, numerals, fixed compounds, or forms that "look right" only by analogy.

## Working Rule

Check orthography separately from grammar. A sentence can have correct case and agreement while still using a nonstandard spelling, misplaced hyphen, or wrong compound form.

For joined, separate, and hyphenated writing, use `compound-writing-rules.md` first. For particles and fixed expressions, use `particle-expression-rules.md`. This file is a compact orthography reminder, not the whole decision system.

## High-Risk Areas

- Hyphenated compounds:
  - Prefer `ორკილო-ნახევარი`.
  - Avoid `ორკილონახევარი`, `ორ კილონახევარი`, `ორ-კილონახევარი`.
- Joined fixed adverbs and particles:
  - Prefer `მაინცდამაინც`.
  - Keep established separate expressions separate when standard, e.g. `ბოლოს და ბოლოს`.
- Compound adjectives and lexicalized forms:
  - Prefer `იმედგაცრუებული`.
  - Avoid `იმედ-გაცრუებული`, `იმედ გაცრუებული`, `იმედგაცრუვებული`.
- Standard verb stems and prefixes:
  - Prefer `ვუგულებელყოფ`.
  - Avoid `ვუგულებელვყოფ`, `უგულებელვყოფ`, `უგულებელვჰყოფ`.
  - Prefer modern standard forms such as `დგას`, `ჩანს`, `წყევლის` unless an archaic style is explicitly requested.
- Directional pronoun forms with `-კენ`:
  - Prefer `ჩემკენ`, `შენკენ`, `ჩვენკენ`, `თქვენკენ`.
  - Avoid `ჩემსკენ`, `შენსკენ`, `ჩვენსკენ`, `თქვენსკენ`.

## Decision Protocol

1. Identify whether the issue is grammar or orthography.
2. For orthography, check fixed examples before applying a general analogy.
3. If no local rule covers the form and uncertainty remains, prefer a safe rephrasing or use a normative external source as a final verification layer.
